# GLITCH://PROTOCOL

A Phaser 3 browser game for the GLITCHED GAMES hackathon.

## Run Locally

Open `index.html` in a browser or serve the folder with a static server.

## Controls

- Move: `A/D` or `Left/Right`
- Jump: `W`, `Space`, or `Up`
- Glitch action: `Shift` or `[!]`
- Pause: `Esc`, `P`, or `⏸`

## Notes

- The game uses only Phaser graphics primitives and one supplied logo image.
- Progress is stored in `localStorage` under `glitch_protocol_save`.
